# LUWI Foundation Website
This is the official website of LUWI Foundation, built with HTML, CSS, and JavaScript.
- Home page with hero banner and testimonials
- About page with video section
- Donation page with amount selection and mobile money input
- Contact page with form
## Deployment
You can deploy this website on Netlify, Vercel, or any hosting platform. Just upload the LUWI-Foundation folder.
